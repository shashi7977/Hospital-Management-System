# Hospital-Management-System
---------------------------------------HOSPITAL MANAGEMENT SYSTEM-----------------------------------
Hospital Management System is a Java JDBC application that helps hospitals and healthcare facilities manage their operations. This system manages patient, doctor records and assists in scheduling appointments
Features:
Patient Management: Maintain records of patients, including personal details and medical 
Doctor Management: Manage records of doctors, including speciality
Appointment Management: Schedule Date and appointments for patients
Database Connectivity: Interact with a MySQL database using Java JDBC

Technology:
Language: Java
Database Connectivity: JDBC
Database: MySQL
